# 🎮 Guia de Uso — CMO 360° Command Center

O **CMO 360° Command Center** é a sua interface de combate diário. Ele consolida todos os motores de marketing em um único painel interativo.

## 🚀 Como Iniciar

Basta dar um duplo clique no arquivo:
`cmo_command_center.bat` na raiz do projeto.

## 📊 Funcionalidades do Painel

### [1] Dashboard Geral

Resumo de alta performance (Receita, Investimento, ROAS e CAC). Use para sua revisão matinal rápida.

### [2] Alertas Críticos

Listagem de anomalias detectadas pelos motores Python (Z-Score) em tempo real.

### [3] Insights da IA

Exibe as últimas recomendações estratégicas geradas pelo modelo **Primary (GPT-4 Turbo)** ou análises de rotina do modelo **Support (Gemini Flash)**.

### [7] Trocar Tenant

Se você gerencia múltiplas marcas ou franquias, pode alternar o contexto de dados do painel instantaneamente.

## 💡 Rotina Recomendada (5 Minutos)

1. **Abrir**: Duplo clique no `.bat`.
2. **Alertas [2]**: Verificar se há algo urgente (Campanhas pausadas, ROAS baixo).
3. **Insights [3]**: Ler a recomendação estratégica do dia.
4. **Dashboard [1]**: Validar se os números estão dentro da meta.
5. **Fechar [0]**: Sair e focar na execução.
