# Workflow 100x — Soberania de Resposta

Este workflow é ativado automaticamente quando o comando contém `100x`. Ele exige que o agente saia do rastro estatístico provável e explore o espaço de exceção.

## 🔄 Passagens Obrigatórias

1. **Passagem 1 (Rascunho)**: Resposta direta e óbvia.
2. **Passagem 2 (Socratic Audit)**:
   - Verificar pressupostos ocultos.
   - Aplicar Socratic Gate v2 (Premissa, Diferença, Simplicidade, Pior Cenário).
   - Identificar Gaps de Visibilidade.
3. **Passagem 3 (Adversarial)**: Cross-domain challenge.
4. **Passagem 4 (Amplifier)**: Sumarizar a visão dos 100 especialistas agrupados por domínio.
5. **Passagem 5 (Convergence)**: Síntese final soberana + @skills.

## 📦 Formato de Saída

```markdown
⟳ REVISÃO 100x ATIVA (3+2 Passagens)

### 🗣️ AMPLIFICADOR: VISÃO DOS 100 ESPECIALISTAS
[Resumo da inteligência coletiva agrupada]

### [RESPOSTA DESTILADA: DIAGNÓSTICO SOBERANO]
[Conteúdo final aqui]

### 🔍 GAPS IDENTIFICADOS
[O que não temos visibilidade]

### ⚖️ DELTA DE SOBERANIA
[Diferença real entre o óbvio e o soberano]
```

## 🚫 Regras

- Zero alucinação.
- Se o dado não existe, declare o gap.
- Uma única resposta, porém densa.
