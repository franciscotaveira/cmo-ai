# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [6.1.0] - 2026-03-02

### ✨ Adicionado
- **Notification Dispatcher** — Sistema de notificações sob demanda
- **E-mail Templates** — Templates HTML profissionais (Alert, Daily Digest, Weekly Summary)
- **Test Notifications** — Script de teste para notificações
- Suporte a múltiplos canais (E-mail, Slack, Telegram, WhatsApp, Push)

### 🔧 Melhorias
- Notificações só são enviadas quando necessário (sem processos 24/7)
- Custo zero de notificações (SMTP grátis)
- Templates responsivos e profissionais

### 📚 Documentação
- `CONFIG_EMAIL.md` — Guia de configuração de e-mail
- `NOTIFICACOES_EMAIL_CONFIGURADAS.md` — Resumo completo

## [6.0.0] - 2026-03-02

### ✨ Adicionado
- **CMO-Bench** — Sistema de aprendizado tipo SWE-bench
- **Knowledge Base** — Base de casos resolvidos
- **Verification System** — Verificação executável de ações
- **Similarity Retrieval** — Busca de casos similares

### 🔧 Melhorias
- Sistema agora aprende com casos passados
- Verificação de resultados (tipo testes unitários)
- Feedback loop automático

### 📚 Documentação
- `CMO_BENCH_INTELIGENCIA.md` — Documentação completa do CMO-Bench
