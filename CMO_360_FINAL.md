## 🚀 COMO USAR

### 1. Setup Inicial (30 minutos)

```bash
# 1. Configurar .env
cd c:\Users\Marketing\Documents\Antigravity\antigravity-kit
cp mkt/.env.example mkt/.env

# Editar .env com:
# - SUPABASE_URL, SUPABASE_KEY
# - GROQ_API_KEY (ou outro LLM)
# - PATH_TO_DRIVE
# - PATH_TO_OBSIDIAN

# 2. Instalar dependências
pip install -r mkt/engine/requirements.txt

# 3. Executar
python -m mkt.engine.main
```

### 2. Acessar no Obsidian

Após execução, abra o Obsidian e navegue para:

